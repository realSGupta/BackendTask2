# Quick Setup Commands

# After creating your GitHub repo, run these commands:

# 1. Add your GitHub repo as remote
# Replace YOUR_USERNAME and YOUR_REPO_NAME with your actual values
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# 2. Rename branch to main (if needed)
git branch -M main

# 3. Push your code
git push -u origin main

# You're done! Your code is now on GitHub.

# To verify:
# Visit: https://github.com/YOUR_USERNAME/YOUR_REPO_NAME
